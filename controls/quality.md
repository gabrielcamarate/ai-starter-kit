# Guardrails de Qualidade

## Definition of Done por mudança
- Código compila e tipa sem erros.
- Testes relevantes atualizados e passando.
- Documentação impactada atualizada.
- Segurança mínima revisada.

## Padrão de PR
- Escopo claro.
- Risco e mitigação descritos.
- Evidências de teste incluídas.
- Se `requires_docs=true`, incluir evidência de Documentation Research.
