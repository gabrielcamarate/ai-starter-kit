# Skill: doc-first-context7

## Objetivo
Exemplo de aplicação do fluxo Doc-first com Context7 para reduzir erro de integração.

## Procedimento passo a passo
1. Consultar `skills/registry.json`.
2. Verificar `requires_docs`.
3. Pesquisar documentação com Context7 (ou fallback oficial quando indisponível).
4. Registrar evidências:
   - Sources
   - Key constraints
   - Version (quando aplicável)
5. Iniciar implementação apenas após registrar a pesquisa.
