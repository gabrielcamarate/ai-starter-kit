# ai-starter-kit

Starter kit profissional para criar projetos novos com padrão consistente e auditar projetos existentes para retrofit seguro.

## Requisitos
- Node.js >= 18
- PNPM

## Instalação
```bash
pnpm install
```

## Comando principal
```bash
pnpm askit --help
```

## Fluxo rápido
```bash
pnpm askit init meu-projeto --template react-ts
pnpm askit audit --path ./meu-projeto
```

## Documentação
- `DESIGN.md`: contrato arquitetural completo
- `docs/QUICKSTART.md`: uso em 5 minutos
- `docs/RETROFIT_GUIDE.md`: aplicar padrão em projetos existentes
- `GITHUB_PUBLISH.md`: publicar no GitHub
