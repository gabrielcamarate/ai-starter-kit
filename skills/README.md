# Skills

Skills são instruções operacionais reutilizáveis para tarefas recorrentes.

## Como escrever
1. Comece com objetivo e escopo.
2. Liste pré-condições.
3. Escreva procedimento passo a passo.
4. Defina critérios de saída e anti-padrões.

Use os modelos em `skills/templates/` e exemplos em `skills/examples/`.
